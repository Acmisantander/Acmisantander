Enviar manuscrito

Recepción de manuscritos

Formulario institucional para el envío de manuscritos académicos, columnas de opinión, editoriales o artículos para el Magazine ACMI Santander.

Puede adjuntar archivos en PDF, DOC, DOCX, RTF, ODT, PPT o PPTX. Si el archivo supera el límite máximo de 100 MB, debe utilizar las opciones alternativas a continuación.

Opciones alternativas para archivos grandes u otros métodos de envío

Si el archivo supera los 100 MB, puede utilizar cualquiera de las siguientes alternativas:

Subir archivo a Google Drive (opcional)

Subir manuscrito a Google Drive

También puede enviar el manuscrito directamente al correo: santanderacmi@gmail.com
