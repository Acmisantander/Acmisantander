Enviar manuscrito

Recepción de manuscritos

Formulario institucional para el envío de manuscritos académicos, columnas de opinión, editoriales o artículos para el Magazine ACMI Santander.

Formatos permitidos: PDF, DOC, DOCX, RTF, ODT, PPT o PPTX. Para archivos mayores de 100 MB, use el botón de Google Drive como respaldo.

Respaldo para archivos grandes:

Si el cargue directo falla o el archivo es muy pesado, súbalo a la carpeta institucional de Google Drive y luego envíe este formulario.
